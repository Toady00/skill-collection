# Test Writing Skill

It's been my experience that agents like to just write tests regardless of the
value of the test. This will change with different models, but I feel providing
some guidance specific to the technology stack being tested can go a long way.
The guides provided are all AI generated based on what the agent at the time
thought was the best practice for that tech stack. This skill needs additional
validation before heavy use.

## Progressive Disclosure

This skill is meant to take heavy advantage of progressive disclosure,
essentially providing very little guidance in the skill itself and allowing the
agent to find the relevant specific guidance from the included guides.
